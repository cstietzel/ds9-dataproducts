Overview

This application allows you to explore the Class of 2015 at public schools in the state of Connecticut. You can view the Graduation Rate or College Entry Rate as a function of the following variables: Absentee Rate, Total District Spend per Pupil, or Total School Enrollment.

Data Selection

The selection high school outcomes and variables is done using the drop down boxes on the left-hand input panel.

Outcomes
  • Graduation Rate: The percentage of students in the Class of 2015 cohort that graduated in four years
  • College Entry Rate: The percentage of students in the Class of 2015 cohort that entered a post secondary school
Variables
  • Absentee Rate: Average absentee rate for school in academic years 2011/12 to 2014/15
  • Per Pupil Spend: Average spend in $'s on each pupil in the school's district in academic years 2011/12 to 2014/15
  • Enrollment: Average total school enrollment in academic years 2011/12 to 2014/15

Output Tabs

Each of the tabs in the application show aspects of the data based on the selected Outcome and Variable. Click on the the tab to show the results.

Scatterplot

A scatterplot of the items selected is displayed. In addition a linear model using the two selections is overlayed that shows the relationship between the outcome and the selected variable as well as the 95% error round the least squares fit. Mousing over a point with display the values for the outcome and variable

Distribution

Histograms of the data for the outcome and variable selected are displayed.

Model Summary

The summary output for the linear model is shown. The model is produced using the lm function. This is the standard R output that shows the coefficient values for the intercept and selected variable as well as p-values for each.

Data

This tab shows the complete data set used for the application. The search box can be used to find schools or districts. Each column can be sorted by clicking on the column heading.
